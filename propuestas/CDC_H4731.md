# #CDC #H4731 App para comerciales

Los objetivos de este proyecto son:

+ Hacer que los comerciales dispongan de una aplicación que les permita generar pedidos desde el navegador de sus teléfonos / tablets / portátiles.

## Propuesta realizada a petición de:
Pablo

## Estructura

### Grupos y permisos.
Definiremos grupos de usuarios con los siguientes permisos:

+ Admin. Acceso total.
+ TPV. Acceso a TPV (ventas y arqueos)
+ Comercial. Acceso a Pedidos (del agente), Clientes (del agente, solo consulta), Artículos (solo para consulta de datos)

### App de pedidos. Pedidos
El apartado pedidos permitirá al agente:

    + Consultar los pedidos de sus clientes
    + Crear y modificar pedidos de sus clientes
    + Copiar pedidos de sus clientes para crear un pedido nuevo a partir de la copia

### App de pedidos. Pedidos. Rescate de productos del histórico
Bajo la lista de líneas de pedido incluiremos una tabla de histórico con los últimos productos comprados por el cliente que permitirá al agente seleccionar registros y crear líneas con la misma referencia y la cantidad indicada.
    
### App de pedidos. Clientes
El apartado clientes permitirá al agente:

    + Consultar los datos de sus clientes, incluyendo:
        + Descuentos por producto

    + Consultar el histórico de pedidos, en una tabla:
        + Referencia y Artículo
        + Cantidad
        + Precio unitario
        + Descuentos
        + Precio total

El apartado clientes permitirá al admin:
    + Crear, modificar y borrar descuentos por producto

### App de pedidos. Artículos
El apartado artículos permitirá al agente:

    + Consultar datos, incluyendo
        + Descuentos por cliente (solo sus clientes)

El apartado clientes permitirá al admin:
    + Crear, modificar y borrar descuentos por cliente
