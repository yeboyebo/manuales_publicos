# #DTS #H4695 Importación de datos para generar asientos y remesas de nóminas

Los objetivos de este proyecto son:

+ Poder importar los datos de nóminas que la asesoría nos proporciona para generar las correspondientes nóminas y sus asientos
+ Disponer de una funcionalidad de generación de remesa de nóminas para enviar al banco

### Recursos Humanos / Importación de Nóminas

En el formulario de nóminas, incluiremos un botón para importar un fichero csv o pegar desde hoja de cálculo.

Las columnas a pegar son:

+ Código de empleado
+ Bruto (`DUDA: columna devengado o columna sueldos`). Cuenta 64000000 Sueldos y Salarios. Debe
+ SS. Trabajador (columna S.S. Trab) 47600000	Organismos S.S. Acreedores. Haber
+ IRPF (columna IRPF). 47510000	H.P. Acree. Retenc. Praticadas (IRPF). Haber
+ Líquido (columna A Percibir). 46500000. Remuneraciones pendientes de pago. Haber

El proceso:
    Comprobará que las nóminas cuadran (bruto = IRPF + SS + Líquido)
    Generará la nómina y su asiento correspondiente

### Tesorería / Remesas de pago
Se instalará la extensión de remesas de pago de nóminas para poder incluir las nóminas y asociarlas a un cuaderno de pago.
