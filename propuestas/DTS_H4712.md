# #DTS #H4712 Exportación de listados de nóminas

Los objetivos de este proyecto son:

+ Exportar los datos de nóminas de un periodo determinado para poder usarlos al cumplimentar informes requeridos por el grupo de empresas.

## Propuesta realizada a petición de:
Roberto

## Estructura

### Recursos Humanos / Empleados
Añadiremos a la tabla de empleados los siguientes campos

+ Género. Hombre / Mujer (ya existe).

+ Categoría. Posibles valores.
    + Dirección
    + Mando intermedio
    + Resto

+ Tipo de contrato
    + Indefinido
    + Temporal
    + Otro

+ Tipo de jornada. Valor entre 0 y 100 que define si el trabajador está a jornada completa (100), media jornada (50), etc.
    

### Recursos Humanos / Informe (opción A)
Crearemos un informe en formato de hoja de cálculo con las siguientes características:

Criterios de búsqueda:
+ Empresa
+ Intervalo de fechas de nómina desde - hasta

Datos a mostrar por cada empleado
    + Nombre
    + Género
    + Tramo de edad. En función de la fecha de nacimiento, < 30, 30-50, > 50
    + Categoría
    + Tipo de contrato
    + Tipo de jornada
    + Salario medio (media de bruto de nóminas en el período)


### Recursos Humanos / Cubo (opción B)
Crearemos un cubo con las siguientes características:

Dimensiones:
+ Trabajador
+ Género
+ Tramo de edad. En función de la fecha de nacimiento, < 30, 30-50, > 50
+ Categoría
+ Tipo de contrato
+ Tipo de jornada

Medidas:
+ Conteo
+ Salario bruto
