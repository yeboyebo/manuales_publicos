# #EUR #H4637 Parametrizar la opción de permitir compensaciones de recibos

Los objetivos de este proyecto son:

+ Poder parametrizar en función de la unidad de negocio o el cliente si se ofrecerá al usuario la opción de compensar recibos en el momento de emitir la factura.


## Propuesta realizada a petición de:
José R.

## Estructura

### Unidades de negocio
Añadiremos a la tabla y el formulario de unidades de negocio el siguiente campo:

* Permitir compensaciones. Posibles valores:
    + Sí (valor por defecto)
    + No

### Clientes
Añadiremos a la tabla y el formulario de clientes el siguiente campo:

* Permitir compensaciones. Posibles valores:
    + Sin definir (valor por defecto)
    + Sí
    + No

Lo añadiremos en la pestaña Comercial, a la derecha del campo de unidad de negocio

+ Campo. Descripción.

### Lógica para mostrar el formulario de compensaciones

| Cliente  | U.N.    | Mostrar compensaciones |
| -------- | ------- | ---------------------- |
| Sí       | *       | Sí                     |
| No       | *       | No                     |
| Sin def. | Sí      | Sí                     |
| Sin def. | No      | No                     |
