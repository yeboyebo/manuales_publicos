# #EUR #H4705 Nuevos avisos para clientes bloqueados en presupuestos y pedidos

Los objetivos de este proyecto son:

+ Mejorar la información y enriquecer los avisos que se muestran al usuario cuando un cliente está bloqueado.

## Propuesta realizada a petición de:
José

## Estructura

### Prespuestos de cliente
Añadiremos a la pantalla la siguiente funcionalidad:

    - Al guardar el presupuesto debe avisar hayan cambiado cosas o no, poner ese mismo aviso "A" y que deje continuar como hace el aviso "A", 

    - Al aprobar el presupuesto (el botón fuera), poner el aviso pero con ¿Desea continuar? Si/no, Llamémosle para aclararnos aviso "C" de confirmar, 
        · Respuesta Si: hace lo de ahora
        · Respuesta No: no genera el pedido y se queda el presupuesto pte de aprobar como si no hubiese pasado nada.

    - Al abrir un presupuesto existente, que dé el aviso "A" y luego abra el presupuesto y deje continuar.


Avisos:

    + "A": El cliente [código de cliente] está bloqueado
    + "C": El cliente [código de cliente] está bloqueado. ¿Desea continuar?

### Pedidos de cliente
Añadiremos a la pantalla la siguiente funcionalidad:
            
    - Al guardar el pedido , debe avisar hayan cambiado cosas o no.  poner ese mismo aviso "A" y que deje continuar como hace el aviso "A", 

    - Al abrir un pedido existente, que dé el aviso "A" y luego abra el presupuesto y deje continuar.
