# #GUA #H4548 Envío de artículos inhabilitados y habilitación masiva posterior

Los objetivos de este proyecto son:
+ Poder configurar los artículos en Magento antes de ser publicados para evitar cuellos de botella en la sincronización y adaptación de artículos a su venta web

## Propuesta realizada a petición de:
Almudena

## Estructura

### ERP / Almacén / Artículos
Modificaremos los actuales campos de Habilitado / Visibilidad / Habilitado hijos / Visibilidad para simplificarlos en un único campo _Habilitado_. Con valor por defecto No.

### ERP / Almacén / Cambios masivos
Modificaremos la pantalla para permitir habiliar / deshabilitar conjuntos de productos

### ERP - Magento / Sincro actual de productos
Modificaremos la sincronización actual de productos para los nuevos productos creados en Magento desde el ERP estén por defecto deshabilitados

### ERP - Magento / Sincro de habilitación de productos
Crearemos una nueva sincronización para habilitar / deshabilitar productos
