## Estructura

### Familia / Tipos de atributo
Añadiremos a la pestaña de tipos de atributo una tabla al lado de la actual tabla de tipos de atributo que mostrará las siguientes columnas:

+ Referencia. Referencia del artículo
+ Descripción. Descripcion del artículo
+ Id. Valor: Id del Valor del tipo de atributo seleccionado en la tabla de tipos de atributo.

## Notas de desarrollo
Usar FLTableDB con vista, filtrada según el tipo de atributo seleccionado.

Vista con:
```sql
select a.referencia, a.descripcion, taf.idtipoatributo, ta.nombre, vaa.idvaloratr, vta.valor FROM articulos a INNER JOIN familias f ON a.codfamilia = f.codfamilia INNER JOIN af_tiposatributofamilia taf ON f.codfamilia = taf.codfamilia
INNER JOIN af_tiposatributo ta ON taf.idtipoatributo = ta.idtipoatributo LEFT OUTER JOIN af_valoresatributoarticulo vaa ON taf.idtipoatributo = vaa.idtipoatributo AND a.referencia = vaa.referencia LEFT OUTER JOIN af_valorestipoatributo vta ON vaa.idvaloratr = vta.idvaloratr 
```

Filtro:
```sql
where taf.idtipoatributo = '1' and f.codfamilia = '38';
```
