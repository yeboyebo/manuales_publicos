# #HFF #H4565 Desactivación de ciertos componentes para el cálculo de inventario valorado

Los objetivos de este proyecto son:
+ Que la valoración de inventario sea realista al usar solo uno de X posibles componentes cuando en una composición hay varias opciones mutuamente exclusivas

## Propuesta realizada a petición de:
Belén y Ana

## Estructura

### Artículos / Composición
Incluiremos en la tabla de composición un nuevo campo
+ Excluir de valoración de inventario. Sí / No. Valor por defecto No.

### Informes / Inventario valorado
Modificaremos el proceso de cálculo de costes para el informe de inventario valorado de forma que se excluya aquellos componentes marcados con el nuevo campo.
