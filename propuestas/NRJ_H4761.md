# #NRJ #H4761 inhabilitación de calibres y filtro de variedades en informe de pesadas

Los objetivos de este proyecto son:

+ Permitir inhabilitar calibres que ya no se usan para que a partir de ahora no aparezcan como opciones en Eneboo, de forma similar a la inhabilitación de los tipos de palets de mercado.
+ Incluir un nuevo criterio de lista de variedades para que un usuario pueda seleccionar varias variedades como filtro en el informe de pesadas.

## Propuesta realizada a petición de:
Rosa

## Estructura

### Almacén / Calibres inhabilitados
Añadiremos a la pantalla los siguientes controles:

+ Habilitado. Valor por defecto Sí. Indica si el calibre está habilitado.

En el formulario maestro de calibres incluiremos un filtro Habilitado = Sí por defecto y un check Mostrar inhabilitados para poder ver los inhabilitados (por ejemplo en el caso de que se desee rehabilitar un calibre previamente inhabilitado)

### Informe / Filtro de variedades en informe de pesadas
Añadiremos a la pantalla de filtros los siguientes controles:

+ Variedades + Botón seleccionar

Al pulsar el botón aparecerá el formulario de lista de variedades en modo multiselección. El usuario seleccionará las variedades que quiera mostrar en el informe y lo guardará.

Al emitirse un informe con el criterio de variedades informado, el filtro incluirá solo las variedades indicadas.
