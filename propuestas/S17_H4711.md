# #S17 #H4711 Importación de facturas desde documentos PDF

Los objetivos de este proyecto son:

+ Facilitar el registro y contabilización de facturas recibidas importándolas directamente desde los ficheros PDF recibidos directamente de los clientes.

## Propuesta realizada a petición de:
Nerea

## Estructura

### Facturación / Importación de borradores de facturas de compra. Estructura

Añadiremos al menú de facturación una opción de _Borradores de factura de compra_ con una estructura similar a la de facturas de compra. El formulario tendrá un botón _Importar desde PDF_ que permitirá seleccionar un fichero PDF y generar una factura asociada.

Añadiremos a la tabla de facturas los siguientes campos

+ Estado importación. Posibles valores:
    + Borrador. La factura se ha importado pero no ha sido validada. 
    + Emitida. La factura ha sido validada, y tiene una factura de compra asociada

### Facturación / Borradores de factura de compra. Importación de PDF
Usaremos un sistema IA LLM (tipo ChatGPT) para leer el PDF e interpretarlo. Los pasos para importar la factura serán:
    + Intentar identificar el proveedor por CIF y nombre
        + Si hay coincidencia, usar los datos de la tabla de proveedores
        + Si no hay coincidencia, dejar el código de proveedor vacío e informar nombre y CIF

    + Ejercicio:
        + Obtendremos el ejercicio de la fecha de la factura
    
    + Para cada línea de la factura
        + Intentar identificar producto por referencia (buscando en referencia de proveedor de la tabal de la pestaña Compras de artículos)
            + Si hay coincidencia, usar referencia y nombre de la tabla
            + Si no hay coincidencia, dejar la referencia de compra vacía e informar solo descripción

        + Intentar buscar el IVA correspondiente del ERP que coincida en tipo con el IVA del PDF
            + Si hay coincidencia, usar código de impuesto y tipo de IVA
            + Si no hay coincidencia, usar solo tipo y avisar.

    + La importación comprobará, y avisará mediante una etiqueta al usuario, si no se cumplen que:
        + La suma de líneas coincide con el total
        + La aritmética de las líneas es correcta (unidades x precio - descuentos = total)
        + El IVA está correctamente aplicado

### Facturación / Facturas de compra. Gestión documental
Asociaremos la gestión documental a las facturas de compra y a los borradores de factura de compra, para que el usuario pueda consultar el documento real a la hora de corregir manualmente el borrador, cuando esto sea necesario.
