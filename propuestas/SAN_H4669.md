# #SAN #H4669 Agilizar recepción de preparaciones / recepciones parciales

Los objetivos de este proyecto son:

+ Agilizar las preparaciones / recepciones eliminando la necesidad de cerrar una a una todas las líneas que no se van a preparar o recibir en casos parciales de preparación o recepción.

## Propuesta realizada a petición de:
Verónica

## Estructura

### Webapp / Preparaciones

Añadiremos a la pantalla un botón global (no asociado a ninguna línea):

+ Cerrar (icono candado). Tras una confirmación cierra todas las líneas no preparadas o servidas parcialmente, habiliando el botón de listo PDA.

### Webapp / Recepciones

Mismo funcionamiento que Webapp / Preparaciones.
