# #SAN #H4687 Informe de tareas con ubicación de finalización

Los objetivos de este proyecto son:
+ Conocer la ubicación (dirección) en la que las tareas han sido terminadas

## Estructura

### Webapp / Informe de tareas por agente
Añadiremos a la pantalla los siguientes controles:

+ Campo. Descripción.

### Webapp / Finalización de tareas
Modificaremos el proceso de finalización de tareas para lanzar un evento _TareaFinalizada_ que ligaremos con una cola asociada a un nuevo proceso _Obtener geolocalización inversa_

### Servidor / Geolocalización inversa
Crearemos un proceso que consumirá eventos de tipo _TareaFinalizada_ para llamar a la API de Google Maps para obtener y guardar la dirección correspondiente a los datos de latitud y longitud registrados en el momento de la finalización de la tarea.

### Servidor / Carga inicial de tareas ya terminadas
Crearemos un proceso que genere los eventos _TareaFinalizada_ correspondiente a todas las tareas finalizadas hasta la fecha para su proceso automático.

## Notas de desarrollo
Es necesario el uso de una clave API Key de Google para que el proceso de geolocalización inversa pueda funcionar.
