# #SAN #H4729 Checkbox en líneas de albaranes para albaranes de curso "Promoción curso", visible solo para cursos

Los objetivos de este proyecto son:

+ Distinguir los materiales que están en promoción dentro de los albaranes que pertenecen a módulos de cursos

## Propuesta realizada a petición de:
Jesús

## Estructura

### Cursos / Módulos / Albaranes
Añadiremos las líneas de albarán de salida un nuevo campo:

+ Promoción curso. Sí / No. Valor por defecto _No_.

Este campo solo estará visible cuando el albarán pertenezca a un módulo de curso.

## Tests

+ Visible cuando el albarán está asociado a un curso
+ No visible cuando el albarán no está asociado a un curso
