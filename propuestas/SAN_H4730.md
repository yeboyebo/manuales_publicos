# #SAN #H4730 Poder asociar facturas de venta a clientes distinto del albarán que la origina

Los objetivos de este proyecto son:

+ Permitir que las facturas estén asociadas a clientes distintos al del albarán que las origina. Pensado para el caso de facturación a empresas de renting.

## Propuesta realizada a petición de:
Jesús

## Estructura

### Facturación / Facturas
Añadiremos a la pantalla los siguientes controles:

+ Botón desbloquear cliente. Al pulsar el botón el campo cliente se habilitará, permitiendo cambiar el cliente.

El botón solo se habilitará si la factura no está presentada en el SII.

El botón no se verá para facturas no automáticas (que no vienen de albarán)
