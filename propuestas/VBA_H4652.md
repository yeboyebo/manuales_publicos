### Artículos / Alta de artículo
Crearemos dos nuevos controles en el formulario de artículos que estarán activos solo cuando se está creando el artículo (modo insert):

+ Check _Formato libre_: Indica que el formato de referencia y descripción será manual, sin calcularse.
+ Etiqueta: Formato de referencia. Posibles valores:
    + Libre, cuando el check _Formato libre_ esté activado.
    + Árbol copado, cuando se indica perímero
    + Palmera, cuando se indica tronco y no perímetro
    + Otros (aromática, etc.), en cualquier otro caso

Cálculo según formato de referencia:

    + Árbol copado:
        + Referencia: Concatenar:
            + 3 primeras letras de nombre1
            + 3 primeras letras de nombre2
            + Iniciales primera y segunda (si existe) de nombre 3. Ejemplo: ACER campestre "queen elizabeth" CTM 10/12 > QE
            + Código de perímetro
            + Código de formato (a menos que el seleccionado tenga activado _Omitir código en referencia_ )
            + Código de estructura
            + Código de forma
            + Código de detalle
        + Descripción: Misma estructura que referencia, indicando las descripciones de cada valor informado en el mismo orden

    + Palmera:
        + Referencia: Concatenar:
            + 3 primeras letras de nombre1
            + 3 primeras letras de nombre2
            + Iniciales primera y segunda (si existe) de nombre 3. Ejemplo: ACER campestre "queen elizabeth" CTM 10/12 > QE
            + Código de tronco
            + Código de formato (a menos que el seleccionado tenga activado _Omitir código en referencia_ )
            + Código de estructura
            + Código de forma
            + Código de detalle
        + Descripción: Misma estructura que referencia, indicando las descripciones de cada valor informado en el mismo orden

    + Otros (aromática, etc.):
        + Referencia: Concatenar:
            + 3 primeras letras de nombre1
            + 3 primeras letras de nombre2
            + Iniciales primera y segunda (si existe) de nombre 3. Ejemplo: ACER campestre "queen elizabeth" CTM 10/12 > QE
            + Código de litraje
            + Código de estructura
            + Código de forma
            + Código de detalle
        + Descripción: Misma estructura que referencia, indicando las descripciones de cada valor informado en el mismo orden       

### Artículos / Copia de artículo
Se modificará el proceso de copia para, en lugar de pedir la nueva referencia, abrir el formulario de alta con los valores del artículo copiado cargados, de forma que el usuario modifique aquellos que cambian y la referencia y descripción del artículo se adapten a dicha modificación.
