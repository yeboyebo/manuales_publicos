# #VBA #H4696 Modificaciones en agupación de albaranes de compra en factura solicitados por Rosa

Los objetivos de este proyecto son:

+ Mejorar la funcionailidad que permite asociar varios albaranes en la misma factura

## Propuesta realizada a petición de:
Rosa

## Estructura

### Compras / Agrupar albaranes / Incluir / Excluir todos
Incluiremos dos botones:

+ Incluir todos
+ Excluir todos

Estos botones permitirán marcar o desmarcar la columna Incluir para todas las filas.

### Compras / Agrupar albaranes / Número de albarán y ordenación
Incluiremos la columna de número de albarán en la tabla

Modificaremos la ordenación para mostrar las filas con el siguiente orden: Fecha, Número de proveedor, Código, Total,  Proveedor y Nombre 

### Compras / Agrupar albaranes / Ver albarán
Incluiremos un botón que permitirá ver el albarán seleccionado
