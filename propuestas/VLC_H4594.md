# #VLC #H4594 Artículos obsoletos y aviso en compras y ventas

Los objetivos de este proyecto son:

+ Poder marcar como obsoletos los artículos que no se usan ya, para no verlos por defecto en el listado de artículos
+ Recibir una advertencia de que se está tratando de comprar / vender un artículo obsoleto en documentos de facturación.

## Propuesta realizada a petición de:
María José

## Estructura

### Almacén / Artículos
Instalaremos la extensión de artículos obsoletos.

Configuraremos la extensión para recibir un aviso en el caso de crear un documento de compras o ventas (presupuestos, pedidos, albaranes o facturas).

## Notas de desarrollo
Refactorizar la extensión para tener dos configuraciones:

+ Artículos obsoletos en ventas:
    + Permitir
    + Avisar
    + No permitir

+ Artículos obsoletos en compras:
    + Permitir
    + Avisar
    + No permitir
