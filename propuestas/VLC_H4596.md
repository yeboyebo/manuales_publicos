# #VLC #H4596 Inclusión de dato de matrículas y transportista en informe de CMR

Los objetivos de este proyecto son:

+ Poder usar el dato de matrículas y empresa de transporte que se informa en el albarán para que dichos datos se visualicen en el informe actual de CMR

Angie

## Estructura

### Albaranes / Formulario
Incluiremos un nuevo campo:

+ Matrícula de remolque. Mismo formato que el actual campo matrícula.

### Albaranes / Informe de CMR
En el apartado 16 (transporte) del informe de CMR incluiremos:

+ Nombre del chófer / empresa de transporte
+ Matrícula
+ Matrícula de remolque.
