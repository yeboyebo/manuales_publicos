# #VLC #H4714 Gestión de stock de carros

Los objetivos de este proyecto son:

+ Controlar cuántos carros y de qué tipo tiene cada transportista.

## Propuesta realizada a petición de:
Angie

## Estructura

### Facturación / Proveedores
Añadiremos a la tabla de proveedores los siguientes campos

+ Es transportista. Sí / No. Valor por defecto _No_.

### Almacén / Documento de Entrega a transportista
Crearemos una nueva opción de menú Entrega a Transportista.

Campos:
+ Id de entrega (00000001, etc)
+ Transportista (código y nombre), obtenido de Proveedores
+ Matrícula
+ Fecha


Formulario de edición de Entrega a transportista

__Albaranes__
Solo en el caso de Salida de producto, incluiremos una lista de albaranes ligado a la entrega, que se podrá seleccionar de la misma forma que se seleccionan los recibos de venta a incluir en una remesa.

El usuario podrá asociar / quitar albaranes de la entrega

__Carros__

También ligada a cada entrega se mostrará una tabla con 5 columnas:

    + Código de tipo de carro
    + Descripción de tipo de carro
    + Cantidad entregada (editable)
    + Cantidad devuelta (editable)
    + Cantidad neta (entregada - devuelta)

Al guardar el documento de entrega, se crearán los correspondientes movimientos de carros, que incrementarán o decrementarán el stock de carros de cada transportista.


### Almacén / Documento de Entrega a transportista. Impresión
Se generará un documento PDF con la plantilla aportada por Angie.

El campo clientes será una lista separada por comas de los nombres de clientes de los albaranes incluidos en la entrega a transportista.

### Almacén / Transportistas
Crearemos una opción de menú llamada Transportistas en las que mostraremos

+ Código de proveedor
+ Nombre de proveedor
+ Lista de stocks por tipo de carro:
    + Tipo de carro (código)
    + Tipo de carro (descripción)
    + Stock. Número entero.

Los registros de stocks por tipo de carro podrán seleccionarse, y se abrirá un formulario de Stock por tipo de carro que mostrará:

    + Tipo de carro (código y descripción)
    + Stock. Número entero
    + Listado de movimientos con:
        + Fecha
        + Cantidad
        + Tipo. Posibles valores:
            + Automático
            + Regularización
        + Código de entrega a transportista (Automático)
        + Motivo de regularización (Regularización)

El formulario de listado de movimientos permitirá crear nuevos movimientos de tipo regularización

La tabla de transportistas no permitirá editar campos del transportista o crear nuevos transportistas, esto debe hacerse desde Proveedores.

### Facturación / Albaranes de venta
Incluiremos un nuevo campo:

+ Entrega a transportista.

En el formulario de edición mostraremos el código de entrega, el nombre del transportista y la fecha de entrega.
