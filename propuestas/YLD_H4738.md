# #YLD #H4738 Actualizar en cabeceras de facturas el valor del campo centro de coste en función de sus líneas

Los objetivos de este proyecto son:

+ Poder ver en el listado de factuas emitidas / recibidas los centros de coste asociados a sus líneas, en caso de que todas las líneas tengan el mismo valor asociado

## Estructura

### Facturación / Facturas de compra
Añadiremos un cálculo al cálculo de totales que informará el campo código de centro de coste.

Se informará el centro si todas las líneas de la factura que tienen centro de coste informado tienen el mismo valor

### Facturación / Facturas de venta

Mismo funcionamiento que en facturas de compra

## Revisión de facturas actuales
Revisaremos las facturas actuales para actualizar el campo de código de centro de coste

UPDATE facturascli SET codcentro = (SELECT MAX(codcentro) FROM lineasfacturascli WHERE idfactura = facturascli.idfactura) WHERE idfactura IN (
    SELECT f.idfactura FROM facturascli f INNER JOIN lineasfacturascli lf ON f.idfactura = lf.idfactura WHERE lf.codcentro IS NOT NULL GROUP BY lf.codcentro HAVING MIN(lf.codcentro) = MAX(lf.codcentro)
)
